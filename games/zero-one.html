0|1 (pronounced simply "zero one") is a small puzzle game where the player moves a cursor around over some nodes and attempts to turn them from 0s into 1s. The game doesn't have an explicit win state, mostly because I couldn't figure out how to implement one with the game logic. Due to this, you can just assume victory when all the 0s have turned into 1s.

This game was made for LOWREZJAM 2022.
